openaq.viz.tsplot¶

openaq.viz.tsplot(data, time_col=None, ax=None, parameter=None, rs='1h', locations=None, plot_kws={}, fmt_axis=True, **kwargs)[source]¶

Plot a timeseries.

Parameters:
  • data (pd.DataFrame) – dataframe with data
  • time_col (string) – column name with time. Defaults to the current index.
  • ax (matplotlib axis) – Plot on ax if you would like.
  • parameter – string with parameter to plot. Can only plot 1 at a time.
